## 清理文件

```sh
# 清理刚刚产生的下载缓存文件
brew cleanup
# 强制清理 n 天以前的缓存文件，包括下载文件以外的缓存
brew cleanup --prune 0
```

## 查看下载文件

```sh
brew --cache
```
