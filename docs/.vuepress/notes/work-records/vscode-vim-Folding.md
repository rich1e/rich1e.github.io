`zc` - 折叠光标所在位置
`zf` - 折叠光标选中段落
`zo` - 打开光标所在位置的折叠

`zM` - 折叠全部内容
`zR` - 打开折叠的全部内容

![[vim-folding.gif]]

`vi{` 选中范围（使用了 `vim-surround`），`af` 扩大选择范围，直到指定内容全部选中， `zf` 折叠选中内容，`zo` 打开折叠内容。
